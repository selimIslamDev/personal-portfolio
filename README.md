# 🚀 Salim Islam — Personal Portfolio

A modern, fully responsive personal portfolio website built with **React**, **Tailwind CSS v4**, and **Framer Motion**. Features smooth animations, dark/light mode, and a functional contact form.

---

## ✨ Features

- 🌙 Dark / Light mode toggle
- 🎨 Smooth animations with Framer Motion
- 📱 Fully responsive (mobile, tablet, desktop)
- 📩 Functional contact form via Web3Forms
- ⚡ Fast performance with Vite
- 🎯 Clean and modern UI design

---

| Technology | Purpose |

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/salim-khan-3/personal-portfolio

# Navigate to project directory
cd personal-portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```

## 📩 Contact Form Setup

This project uses **Web3Forms** for the contact form.

1. Go to [web3forms.com](https://web3forms.com)
2. Enter your email and get a free **Access Key**
3. Open `src/Components/Contact.jsx`
4. Replace `YOUR_WEB3FORMS_ACCESS_KEY` with your key:

```js
access_key: 'your-access-key-here',
```
